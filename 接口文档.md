## 获取验证码
请求URL：`域名/api/public/?service=Login.getCode`<br>
请求方式：POST<br>

请求参数：

参数名|类型|必选|说明
---|:--:|---:|---:
mobile|string|必填|手机号
sign|string|必填|签名（请求字符串 + 私钥）

返回值：

参数名|类型|说明
---|:--:|---:
code|x|返回错误码
msg|x|错误提示
info|x|数据


```java	
{
    "ret": 200,
    "data": {
        "code": 1001,
        "msg": "签名错误",
        "info": []
    },
    "msg": ""
}
```
